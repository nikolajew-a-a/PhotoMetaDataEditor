# PhotoMetaDataEditor

Desktop-приложение для просмотра фотографий и видео и исправления их метаданных (дата съёмки, геолокация).

## Возможности (план)

- Просмотр фото и видео из папки на диске (без импорта в библиотеку)
- Разделение файлов на «обработанные» и «необработанные» (ручная отметка)
- Редактирование даты съёмки и геолокации (выбор точки на карте)
- Батч-редактирование: сдвиг даты у группы файлов, простановка одной локации на набор
- Карта со всеми геотегированными фотографиями

## Стек

- Kotlin Multiplatform + Compose Multiplatform (desktop сейчас, Android — в планах)
- SQLite (SQLDelight) для хранения статусов обработки
- ExifTool для чтения/записи метаданных фото (EXIF/XMP) и видео (MP4/MOV)
- Leaflet + OpenStreetMap для карты
